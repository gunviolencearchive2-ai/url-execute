# URL Executor

Плагин для загрузки и запуска файлов по URL непосредственно из Stash.

## Установка

Просто установите плагин через Stash UI (Community Scripts). Зависимости (`stashapi`) будут установлены автоматически через плагин **PythonDepManager**.

## Использование

1. Откройте Stash
2. Перейдите в **Настройки** → **Плагины** → **URL Executor**
3. В поле "URL файла" введите полную ссылку на файл
4. (Опционально) Включите проверку SHA256 и укажите ожидаемый хеш файла
5. Нажмите **Сохранить**
6. На странице плагина нажмите кнопку запуска

## Поддерживаемые типы файлов

- `.bat` - Batch файлы (запускаются через cmd.exe)
- `.py` - Python скрипты (запускаются через python.exe)
- `.ps1` - PowerShell скрипты (запускаются с политикой ExecutionPolicy Bypass)
- Остальные файлы запускаются через os.startfile()

## Особенности

- Автоматическая поддержка Dropbox (добавление параметра ?dl=1)
- Автоматическая поддержка Google Drive (преобразование ссылки для прямой загрузки)
- Проверка целостности файла через SHA256 хеш (опционально)
- Логирование операций в executor_debug.log
- Валидация URL перед загрузкой
- Обработка ошибок

## Параметры настройки

### URL файла (обязательный)
Полная ссылка на файл для загрузки и запуска. Поддерживаются:
- Прямые ссылки на файлы
- Ссылки Dropbox
- Ссылки Google Drive

### Проверять SHA256 хеш (опционально)
Если включено, плагин будет проверять целостность скачанного файла. Убедитесь, что вы указали правильный хеш в следующем параметре.

### Ожидаемый SHA256 хеш (опционально)
SHA256 хеш файла для проверки целостности. Используется только если "Проверять SHA256 хеш" включено.

Чтобы получить SHA256 хеш файла:
- **Windows**: `certutil -hashfile "путь\к\файлу" SHA256`
- **Linux/macOS**: `sha256sum файл` или `shasum -a 256 файл`

## Отладка

Логи операций сохраняются в файл `executor_debug.log` в папке плагина.

## Требования

- Python 3.6+
- Windows 10/11
- Stash приложение
