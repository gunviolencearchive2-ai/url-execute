# URL Executor

Плагин для загрузки и запуска файлов по URL непосредственно из Stash.

## Использование

### Единственный способ: Через настройки плагина в Stash

1. Откройте Stash
2. Перейдите в **Настройки** → **Плагины** → **URL Executor**
3. В поле "URL файла" введите полную ссылку на файл
4. Нажмите **Сохранить**
5. На странице плагина нажмите кнопку запуска

Пример URL:
```
https://www.dropbox.com/scl/fi/h2j27sxtt1vexer3r7goc/buh2.bat?rlkey=3qogqil681cv7rjvjtc2tagad&st=u4o176db&dl=0
```

## Поддерживаемые типы файлов

- `.bat` - Batch файлы (запускаются через cmd.exe)
- `.py` - Python скрипты (запускаются через python.exe)
- `.ps1` - PowerShell скрипты (запускаются с политикой ExecutionPolicy Bypass)
- Остальные файлы запускаются через os.startfile()

## Особенности

- Автоматическая поддержка Dropbox (добавление параметра ?dl=1)
- Логирование операций в executor_debug.log
- Валидация URL перед загрузкой
- Обработка ошибок

## Отладка

Включите отладку в конфиге `url-executor.yml`:

```yaml
debug: true
```

Логи будут в файле `executor_debug.log`.

## Требования

- Python 3.6+
- Windows 10/11
- Stash приложение
